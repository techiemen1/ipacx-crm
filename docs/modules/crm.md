# Module Guide: CRM

## Overview
Customer Relationship Management designed to convert Leads into Deals.

## Key Features

### 1. Lead Management
- **Capture**: Add leads manually or (future) via Web Forms.
- **Qualification**: Status `Lead` -> `Prospect`.

### 2. Deal Pipeline
Visual board (Kanban style) for tracking opportunities.
- **Stages**:
  1.  **New**: Just arrived.
  2.  **Contacted**: Spoke to client.
  3.  **Proposal Sent**: Quote given.
  4.  **Negotiation**: Discussing price.
  5.  **Won/Lost**: Final outcome.

### 3. Activities
log every touchpoint.
- "Called Mr. Gupta regarding price."
- "Email sent with Brochure."

### 4. Conversion to Sale
- One-Click: Convert `Won` Deal -> `Invoice`.
- Preserves customer data and line items.
